The desktop app is a native installer — no browser, no login portal to remember. It runs on Windows and macOS.

## Download

Go to the [Busiman downloads section](/#download) and pick your operating system. The link always points at the latest published build.

- **Windows** — downloads a `.exe` installer (NSIS). Run it and follow the prompts; no admin rights are required for a per-user install.
- **macOS** — downloads a `.dmg`. Open it and drag Busiman into your Applications folder.

> **Note:** On Windows, SmartScreen may show an "unknown publisher" warning the first time you run the installer, since the build isn't code-signed yet. Choose **More info → Run anyway**.

## Updating

The desktop app checks for updates on launch. When one is available you'll see a dialog — **Download Now** or **Later** — and the update installs the next time you quit the app. You never need to reinstall by hand. The current version is shown in **Settings → Account**.
